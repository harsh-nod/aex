# AEX VS Code Extension

This lightweight extension provides syntax highlighting and starter snippets for `.aex` task contracts. It recognises the core keywords (`agent`, `goal`, `use`, `do`, `check`, `return`, etc.), supports hash-style comments, and ships with an opinionated scaffold snippet to get started quickly.

The extension is intentionally minimal so it can evolve alongside the reference implementation. Install the packaged `.vsix` or sideload the folder while developing AEX contracts locally.
