# AEX Documentation

Welcome to the AEX documentation. Start here to understand the philosophy, the contract format, and how to integrate it with your agent stack.

## What is AEX?

AEX (Agent Execution Contracts) is a tiny task-contract language for constraining AI agents, tool calls, checks, and approvals. It sits between Markdown instructions and tool execution, giving teams a portable, enforceable layer of policy.
